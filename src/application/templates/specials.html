We all like to think that everything we do is special.

These are some of the special things we are currently spending time on.

 

Presentative

Presentative is a little, adaptive, highly curated group blog project that we
have been working on with some friends. It is barely in alpha status. If you want
to be included in testing/trying out, or if you already know and like us and we
somehow missed you in the initial inclusionary email, let us know.

Neohermes Desktop

The Neohermes Desktop is a graphical, non-verbal Desktop/Operating system based on Lubuntu, designed especially for individuals who have difficulty with reading, writing, or fine motor movement. More info.

Tracker/Keeper

Our Tracker/Keeper is a combination android and web app for tracking and reporting travel and work time and tasks, particularly for those who have concerns related to HIPAA regulations. It is currently in alpha testing and will soon (hopefully) be opened up for beta testers. Let us know if you or your company would like to try it out.

DNA Music

We love music. In fact, we think music is, literally, part of our genes. DNA Music "draws" multi-dimensional non-linear equations (think n-dimensional Sierpinski triangles) based on DNA from selected organisms, then uses these equations to describe multi-dimensional non-linear aural sculptures, i.e., music. Then we save them as .midi files so we can listen at our leisure.

We have been working on this project for a while, and find it promising for several kinds of investigations involving non-linear data structures, including phylogenetics, picture searching, and, not least, making music from the fractal picture of our own genome.

Pyxcel Spreadsheet Writer

We use this utility, written in Python 2.7, to transcribe data into .xls format in our Tracker/Keeper app. It's pretty handy; as soon as we have the time, we will be making it available to everyone, on the Python Package Index.
